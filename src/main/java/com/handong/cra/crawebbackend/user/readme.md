# USER

> 유저 기능 도메인

## 유저정보 수정

1. 유저 비밀번호 변경
2. 유저 기본 정보 변경
3. 유저 프로필 사진 변경

## 로그인시 유저 기능

1. username, password 일치하는 데이터 있는지 확인
2. 회원가입시 username 중복 확인
3. 회원가입
4. 유저 정보 읽어오기
5. 최근 로그인 시각 기록 -> 휴면 계정 처리

### 휴면 계정 구현

1. 1년간 로그인하지 않는다면 휴면 계정 처리 (BaseEntity deleted => true 처리) => 로그인불가
2. 휴면계정 처리 이후 1년 이후 계정 DB에서 삭제

해당 기능은 [cron-bot](../util/Scheduler.java) 으로 구현되어있습니다.
